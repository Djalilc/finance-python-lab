# Stock Price Analyser

A beginner Python project for downloading, analysing, and visualising historical stock price data.

## Purpose

The goal of this project is to learn how to work with real financial data using Python.

This project introduces:

- stock tickers
- historical price data
- closing prices
- daily returns
- basic financial analysis
- data visualisation

## Finance Concepts

A stock represents ownership in a company.

A ticker is the short symbol used to identify a stock, such as:

- AAPL = Apple
- MSFT = Microsoft
- TSLA = Tesla

The main price used in this project is the closing price, which is the price of the stock when the market closes on a given day.

A return measures how much the stock price changed over time.

## Features

The program will:

- ask the user for a stock ticker
- download historical stock price data
- display the first rows of the dataset
- calculate basic price statistics
- calculate daily returns
- plot the stock price over time
- plot daily returns
